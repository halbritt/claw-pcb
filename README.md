## How It Works

```python
board = AltiumProject("openclaw-led-matrix")

# Add nets
VCC = board.add_net("VCC")

# Place components
board.add_pads(smd_rect(x, y, pins, ...))
board.add_component(footprint(...), "ATtiny85", "U1")

# Route traces
board.add_traces(route_90(x1, y1, x2, y2, net=VCC, via=True))

# Add silkscreen
board.add_silk([
    silkscreen_text("OPENCLAW", 20, 4, size=1.5),
    silkscreen_circle(70, 8, 3),  # monkey face
])

# Generate
project = board.build()
json.dump(project, f, indent=2)
```
